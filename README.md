# TransBot
This is a Node.js service that can fetch untranslated messages from translatewiki.net and send translations to them using the Telegram chat network.

Coding conventions
See CONTRIBUTING.md.

# Setting up

1. Run:

```bash
npm install
cd db/
node InitDatabase.js
```
